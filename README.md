# Project Name

This is a [React Native](https://reactnative.dev/) project built with [Expo](https://expo.dev/).

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the application:**
    ```bash
    npm start
    ```
    This will start the Expo development server. You can then run the application on an iOS simulator, Android emulator, or in a web browser.

## Project Structure

The project is organized into the following directories:

*   `app/`: Contains the screens and routing for the application, using Expo Router.
*   `src/`: Contains the core source code, including API clients, custom hooks, utility functions, and more.
*   `components/`: Contains reusable UI components.
*   `assets/`: Contains static assets such as images, fonts, and icons.
*   `constants/`: Contains constant values used throughout the application.

## Available Scripts

*   `npm start`: Starts the Expo development server.
*   `npm run android`: Starts the Expo development server and opens the application on a connected Android device or emulator.
*   `npm run ios`: Starts the Expo development server and opens the application in an iOS simulator.
*   `npm run web`: Starts the Expo development server and opens the application in a web browser.
